D_cache Optimizations:

  • Set Dueling:

Out of order Execution:

  • Modified the current pipeline to scalar OOO processor similar to the one being used by Alpha21264 with Partial Ordering for memory disambiguation policy.
  • Reduced the CPI from the baseline by 13.15%, -9.75%, 2.61%, 9.91%.
